Add to radiation_damage.py (after the existing RadiationDamageParams class and pure functions, before compute_damaged_params):

1. AnnealingParams dataclass with per-defect activation energies and attempt frequencies:
   - E_a_Z12: float = 4.5 eV (carbon vacancy, stable below ~1000C -- see RESEARCH.md open question 4 for calibration rationale; 3.5 eV gives unwanted annealing at 1000C)
   - nu_0_Z12: float = 1e13 Hz
   - E_a_EH67: float = 3.2 eV (stable below ~1200C)
   - nu_0_EH67: float = 1e13 Hz
   - E_a_EH4: float = 1.8 eV (anneals at ~400-600C)
   - nu_0_EH4: float = 1e13 Hz
   - source: str with literature note about confidence levels
   - **post_init** validation: all E_a > 0, all nu_0 > 0

2. annealing_fraction(T, t, E_a, nu_0=1e13) -> float:
   - First-order Arrhenius: f = 1 - exp(-nu_0 _ t _ exp(-E_a / (k_B_eV \* T)))
   - k_B_eV = 8.617e-5 eV/K (use existing constant or define \_K_B_EV at module level)
   - Return 0.0 if t <= 0
   - Clip exponent at 700 to avoid overflow (return 1.0 for exponent > 700)
   - Must accept scalar T and t; use np.exp for consistency
   - Docstring with full formula, parameter descriptions, units

3. defect_recovery_fractions(T_anneal, t_anneal, anneal_params=None) -> dict:
   - Returns {"f_Z12": ..., "f_EH67": ..., "f_EH4": ...} each in [0, 1]
   - Defaults to AnnealingParams() if anneal_params is None

4. compute_annealed_params(pristine_tau_n, pristine_tau_p, N_D_profile, fluence, energy_MeV=62.0, T_anneal=None, t_anneal=None, damage_params=None, anneal_params=None, lifetime_model="linear", T_device=300.0) -> dict:
   - If T_anneal is None or t_anneal is None: return compute_damaged_params(pristine_tau_n, pristine_tau_p, N_D_profile, fluence, energy_MeV, damage_params, lifetime_model, T_device) unchanged (no annealing = pass-through)
   - Otherwise:
     a. Call compute_damaged_params to get irradiated state
     b. Compute per-defect recovery fractions via defect_recovery_fractions(T_anneal, t_anneal, anneal_params)
     c. Reduce defect concentrations: N_X_annealed = N_X \* (1 - f_X) for each defect
     d. CRITICAL: Recompute lifetimes from annealed defect concentrations using the proper physics:
     - Build a temporary RadiationDamageParams with reduced eta values: eta_X_eff = eta_X \* (1 - f_X)
     - Recompute K_tau from these effective etas using compute_K_tau
     - Recompute tau_n, tau_p using degraded_lifetime with the effective K_tau and original fluence_neq
     - Do NOT simply interpolate between pristine and damaged lifetimes (Pitfall 5 from RESEARCH.md)
       e. Carrier removal recovery: scale eta_removal by (1 - f_Z12) since carrier removal is Z1/2-dominated (RESEARCH.md open question 3)
     - Recompute N_D_annealed = apply_carrier_removal(N_D_profile, eta_removal \* (1 - f_Z12), fluence_neq)
       f. Return dict with same keys as compute_damaged_params plus: f_Z12, f_EH67, f_EH4, T_anneal, t_anneal

Update the module docstring to mention annealing kinetics.
Add AnnealingParams, annealing_fraction, defect_recovery_fractions, compute_annealed_params to the module's public API.
