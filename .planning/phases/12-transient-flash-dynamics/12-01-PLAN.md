  <name>Task 1: Implement TransientSolver with pulse envelope and adaptive time-stepping</name>
  <files>src/transient.py</files>
  <action>
Create src/transient.py with the following components:

1. **pulse_envelope(t, t_rise, t_duration, t_fall)** -- standalone function returning envelope value [0,1] for trapezoidal pulse shape. Arguments in seconds. Returns float. Stages: linear ramp 0->1 during t_rise, hold at 1.0 for t_duration, linear ramp 1->0 during t_fall, 0 after.

2. **adaptive_dt(t, t_rise, t_duration, t_fall, dt_min=1e-8, dt_max=1e-4)** -- standalone function selecting time step based on pulse phase. During rise/fall transitions: dt = max(dt_min, t_rise/10). During plateau and post-pulse: dt = dt_max. Returns float.

3. **TransientSolver class** with:
   - `__init__(self, device_info, contact="cathode", method="transient_bdf1")` -- stores device_info, contact, method. Does NOT create the device (caller provides it already biased with Auger).
   - `initialize(self)` -- calls `add_generation_to_dd(device_info, zeros)` then `devsim.solve(type="transient_dc", absolute_error=1e10, relative_error=1e-10, maximum_iterations=30)` to set initial transient state. CRITICAL per Phase 7 pattern.
   - `simulate_pulse(self, G_spatial, t_rise=1e-6, t_duration=1e-3, t_fall=1e-6, t_post=None, dt_min=1e-8, dt_max=1e-4, dose_rate_Gy_s=None)` -- main simulation loop:
     a. If t_post is None, default to 5*t_fall (enough for carrier decay)
     b. t_end = t_rise + t_duration + t_fall + t_post
     c. Record I_dark = extract_contact_current before pulse starts
     d. Time loop: compute envelope, update G_t = G_spatial * envelope via add_generation_to_dd, call devsim.solve(type=method, tdelta=dt, ...), extract current, append to lists
     e. Use logging.info for progress every 50 steps
     f. Return dict: {"times": np.array, "currents": np.array, "I_dark": float, "t_rise": t_rise, "t_duration": t_duration, "t_fall": t_fall, "dose_rate_Gy_s": dose_rate_Gy_s}
   - `compute_transient_cce(self, result, G_spatial, x_nodes)` -- extract CCE from simulation result:
     a. Q_collected = np.trapezoid(|I(t)| - |I_dark|, times) over the pulse duration (t_rise + t_duration + t_fall + t_post)
     b. G_total = np.trapezoid(G_spatial, x_nodes) -- total generation rate per unit area (cm^-2 s^-1)
     c. Q_generated = Q_ELECTRON _ G_total _ t_duration (charge generated during plateau)
     d. CCE = Q_collected / Q_generated, clipped to [0, 2] (allow slight overshoot for transit-time effects)
     e. Return float

Import from existing modules: create_dd_device, ramp_bias, extract_contact_current from drift_diffusion; add_generation_to_dd from charge_collection; add_auger_recombination from flash_recombination; proton_generation_profile from generation_profiles. Use Q_ELECTRON = 1.602e-19 as module constant.

Use CGS units throughout (cm, cm^-3, s, A/cm^2) per project convention.
