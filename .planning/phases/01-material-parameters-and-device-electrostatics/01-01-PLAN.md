    Create `src/incomplete_ionization.py`:
    - Function `ionized_acceptor_fraction(N_A, T=300, E_A=None, g_A=4, k_B=8.617e-5)`:
      - If E_A is None, compute doping-dependent E_A using correction: `E_A_eff = 0.220 - alpha * N_A**(1/3)` where alpha is calibrated so that at N_A=1e19, the ionization fraction falls in 10-30% range. Per RESEARCH.md, this is a calibration task.
      - Calibration approach: use `E_A_eff = 0.220 - 3.4e-8 * N_A**(1/3)` as starting point. At N_A=1e19, N_A^(1/3) = 2.154e6, so correction ~ 0.073 eV, giving E_A_eff ~ 0.147 eV. f_A = 1/(1 + 4*exp(0.147/0.02585)) = 1/(1 + 4*290.7) ~ 0.086% -- still too low.
      - CRITICAL: The simple Gibbs formula cannot reach 10-30% with any reasonable E_A. For N_A ~ 1e19, impurity band formation dominates. Implement a hybrid approach: (1) for N_A < 1e18, use the Gibbs formula with doping-dependent E_A; (2) for N_A >= 1e18, use an empirical model calibrated to literature values. A reasonable calibration: at N_A=1e19, ionization fraction ~ 0.15 (15%). This can be implemented as a smooth interpolation or a lookup.
      - The function must return a float between 0 and 1.
    - Function `ionized_acceptor_concentration(N_A, T=300, **kwargs)`: returns `N_A * ionized_acceptor_fraction(N_A, T, **kwargs)`. This is the N_A^- used in all downstream calculations.
    - Document the physics reasoning and calibration choices in docstrings.

    Create `src/analytical.py`:
    - Function `built_in_potential(N_A_ionized, N_D, n_i, T=300, k_B=8.617e-5)`:
      - Implements V_bi = (kT) * ln(N_A_ionized * N_D / n_i^2)
      - NOTE: kT is in eV, and since q=1 in eV units, V_bi comes out in Volts directly.
      - Must use IONIZED acceptor concentration, not total N_A.
    - Function `depletion_width(V_bi, V_applied, N_D, eps_r=9.7)`:
      - One-sided approximation (valid for N_A >> N_D): W = sqrt(2*eps*(V_bi - V_applied)/(q*N_D))
      - eps_0 = 8.854e-14 F/cm, q = 1.602e-19 C
      - V_applied is negative for reverse bias (so V_bi - V_applied increases W)
      - Add optional `epi_thickness` parameter (default None). If provided, clamp W to epi_thickness (punch-through).
      - Return W in cm.
    - Function `electric_field_profile(x_cm, V_bi, V_applied, N_D, eps_r=9.7)`:
      - Returns E-field array for position array x_cm.
      - In the depletion region (0 to W): E(x) = -q*N_D*(W-x)/eps (triangular profile, max at junction)
      - Outside depletion region: E = 0
      - Return in V/cm.
    - Function `depletion_width_vs_bias(V_biases, V_bi, N_D, eps_r=9.7, epi_thickness=None)`:
      - Convenience function: returns array of W values for an array of bias voltages.
