Create src/microdosimetry.py with the following functions. Follow the existing module docstring pattern from mc_coupling.py (module docstring, logging, clear section headers).

1. **mean_chord_length(sv_thickness_um, sv_width_um=None, sv_depth_um=None)**
   - If all 3 dimensions given: l_bar = 4V/S (Cauchy theorem for rectangular parallelepiped)
   - If only thickness: slab approximation l_bar = 2\*thickness (lateral >> thickness)
   - Returns l_bar in um

2. **make_y_bins(y_min=0.01, y_max=1e4, bins_per_decade=50)**
   - Create 300 log-spaced bins per ICRU 36 convention
   - Use np.logspace for edges, geometric mean (sqrt(e1\*e2)) for centers
   - Return dict with bin_edges, bin_centers, bin_widths (all ndarray)

3. **lineal_energy_spectrum(collected_energies_keV, l_bar_um, y_min=0.01, y_max=1e4, bins_per_decade=50)**
   - Convert energies to lineal energy: y = epsilon / l_bar (keV deposited, l_bar in um -> keV/um)
   - Histogram into log-spaced bins via np.histogram
   - Compute f(y_i) = counts_i / (N_total * dy_i) so integral f(y)*dy = 1
   - Compute y_F = sum(y_i _ f(y_i) _ dy_i)
   - Compute d(y_i) = y_i \* f(y_i) / y_F
   - Compute y_D = sum(y_i _ d(y_i) _ dy_i)
   - Validate: integral f(y)dy approx 1 (warn if > 1%), y_D >= y_F (warn if violated)
   - Handle empty bins gracefully (f=0, d=0)
   - Return dict: bin_edges, bin_centers, bin_widths, f_y, d_y, y_F, y_D, n_events, y_values (raw y per event)

4. **load_stopping_powers(filepath)** (internal helper)
   - Load CSV with columns: energy_MeV, stopping_power_MeV_cm2_per_g
   - Return (energy_MeV array, S array)

5. **compute_kappa_table(water_csv_path, sic_csv_path)**
   - Load both stopping power tables
   - Interpolate to common energy grid using np.interp
   - kappa(E) = S_water(E) / S_SiC(E)
   - Return dict: energy_MeV, kappa

6. **tissue_equivalence_correction(y_values, event_energies_keV, kappa_table=None, kappa_constant=0.58)**
   - If kappa_table provided: interpolate kappa per event energy, y_tissue = kappa \* y_SiC
   - If kappa_table is None: use constant kappa with logged warning about approximation
   - Return corrected y_values array

7. **plot_yd_spectrum(bin_centers, d_y, y_F=None, y_D=None, ax=None, label=None, color=None)**
   - Plot y\*d(y) vs y on semilog-x (standard microdosimetric convention)
   - x-axis: y (keV/um) log scale, y-axis: y\*d(y) linear scale
   - Add vertical lines for y_F (dashed) and y_D (solid) if provided
   - Label axes with units, use matplotlib rcParams for publication quality

8. **plot_yf_spectrum(bin_centers, f_y, y_F=None, ax=None, label=None, color=None)**
   - Plot y\*f(y) vs y on semilog-x
   - Same conventions as plot_yd_spectrum
