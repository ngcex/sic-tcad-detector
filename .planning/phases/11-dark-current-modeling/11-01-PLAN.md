**2. Create src/dark_current.py with these functions:**

Physical constants needed: hbar = 1.0546e-27 erg\*s, m0 = 9.109e-28 g, q = 1.602e-19 C.

**`setup_tat_model(device_info, E_t=None, m_t=None, N_t=None)`**

- Defaults from device_info["params"] if not provided
- Step 1: Create node model `E_field_node` by averaging absolute edge electric field to nodes. Use devsim.edge_average_model() or manually: for interior nodes, average the two adjacent EdgeElectricField values; for boundary nodes, use the single adjacent edge. The EdgeElectricField model already exists from setup_poisson. Compute abs(E_field_node) and clamp minimum to 1e3 V/cm to avoid division by zero.
- Step 2: Compute Kt parameter at each node: `Kt = (4/3) * sqrt(2 * m_t * m0) * (E_t * q_erg)^(3/2) / (q_cgs * hbar * E_field_node)` where q_erg = E_t in eV * 1.602e-12 erg/eV for the energy, and q_cgs = 4.803e-10 statcoulombs. IMPORTANT: Be very careful with CGS vs SI units here. The electric field from devsim is in V/cm. The formula needs consistent units. Recommend computing in SI first (E in V/m, m in kg, E_t in J, hbar in J*s) then the result Kt is dimensionless.
- Step 3: Compute field enhancement factor Gamma_n, Gamma_p as node models. Use the Schenk approximation: for Kt > 0, `Gamma = (sqrt(pi) / (2*Kt)) * exp(1/(4*Kt^2))` when Kt < 4 (high field), and `Gamma = 1` when Kt >= 4 (low field, no enhancement). Use ifelse() in devsim model expressions or compute numerically via get/set_node_model_values.
- Step 4: Create TAT generation rate node model: `U_TAT = (Electrons*Holes - n_i^2) / (taup*(Electrons + n1*Gamma_n) + taun*(Holes + p1*Gamma_p))` where n1, p1 are for the Z1/2 trap level (NOT midgap): `n1 = n_i * exp((E_t_from_midgap)/(kT))`, `p1 = n_i * exp(-(E_t_from_midgap)/(kT))` with E_t_from_midgap = E_g/2 - E_t (trap at Ec - 0.65, midgap at Ec - E_g/2).
- Step 5: Replace the existing ElectronGeneration and HoleGeneration node models with TAT versions: `Gn_TAT = -ElectronCharge * U_TAT`, `Gp_TAT = +ElectronCharge * U_TAT`. Also create derivatives w.r.t. Electrons and Holes for Newton solver convergence.
- IMPORTANT: The derivatives for U_TAT w.r.t. Electrons/Holes must be computed. Since Gamma depends on E_field (which depends on Potential, not carriers), Gamma can be treated as constant for the carrier derivatives. The derivative of U_TAT w.r.t. Electrons is the same form as standard SRH but with Gamma-modified n1, p1.

**`setup_surface_recombination(device_info, S_n=None, S_p=None, contact="cathode")`**

- Defaults from device_info["params"] if not provided
- Add a contact node model for surface recombination current at the specified contact
- `J_SRV = q * S * (n*p - n_i^2) / (S_n*(n + n_i) + S_p*(p + n_i))` simplified for the 1D contact case
- Use devsim.contact_node_model to add this as additional current at the contact
- The sign convention must match the existing contact current extraction

**`extract_dark_current_components(device_info, area=0.05)`**

- Extract J_total from extract_contact_current (A/cm^2)
- Extract J_SRH by evaluating the standard SRH formula at each node and integrating (or store separately during setup)
- J_TAT = J_total - J_SRH - J_SRV (by difference, since they share the same equation structure)
- Actually simpler: compute and store U_SRH_only (standard midgap SRH without field enhancement) as a separate node model during setup_tat_model. Then J_SRH = integrate U_SRH_only _ q over device, J_TAT = integrate (U_TAT - U_SRH_only) _ q over device.
- Return dict: {"J_total": float, "J_SRH": float, "J_TAT": float, "J_SRV": float, "I_total": float (J_total\*area), "I_SRH": ..., "I_TAT": ..., "I_SRV": ...}
- area default 0.05 cm^2 (5 mm^2, typical SiC detector)

**`dark_current_sweep(device_info, V_range, area=0.05, V_step=0.5)`**

- Sweep reverse voltage using incremental ramping (same pattern as iv_sweep)
- At each voltage, call extract_dark_current_components
- Return dict: {"voltages": array, "I_total": array, "I_SRH": array, "I_TAT": array, "I_SRV": array, "J_total": array, ...}

**`create_dark_current_device(T=300, N_t=None, S_n=None, S_p=None, **kwargs)`\*\*

- Convenience function: creates DD device, then calls setup_tat_model and setup_surface_recombination
- Returns device_info (same dict, TAT-enabled)
- Pass through kwargs to create_dd_device (epi_thickness, doping, etc.)
