```python
def plot_cce_heatmap_2d(device_info, cce_map, ax=None, levels=50, cmap="RdYlGn", mirror=True):
    """Plot 2D CCE heatmap on the device mesh.

    Parameters
    ----------
    device_info : dict
        2D device info dict.
    cce_map : array_like
        CCE values at each mesh node (from cce_heatmap_2d).
    ax : matplotlib Axes, optional
    levels : int
        Number of contour levels.
    cmap : str
        Colormap. RdYlGn shows dead (red) -> active (green).
    mirror : bool
        If True, mirror about x=0 to show full device (not just half).

    Returns
    -------
    ax : matplotlib Axes
    """
```

Implementation: use get_triangulation to get (triang, x, y). If mirror=True, create a mirrored copy of x, y, cce_map (negate x, concatenate), rebuild triangulation via Delaunay or matplotlib.tri.Triangulation. Plot with ax.tricontourf. Add colorbar with label "CCE". Convert axes to micrometers. Set axis labels and equal aspect ratio.

For mirroring: simplest approach is to create a new Triangulation from the combined points using `matplotlib.tri.Triangulation(x_full, y_full)` which auto-computes the Delaunay triangulation. This avoids manual triangle index remapping.
