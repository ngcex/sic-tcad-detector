Implementation:

1. Create DD device with create_dd_device(epi_thickness_cm=epi_thickness_cm) using uuid device name
2. Call add_auger_recombination(device_info)
3. Ramp to V_bias using ramp_bias(device_info, V_bias, contact="cathode") -- note cathode for reverse bias per Phase 3 convention
4. Get mesh nodes: x_nodes = devsim.get_node_model_values(device, region, "x")
5. Loop over sorted dose_rates (ascending for continuation stability):
   a. Generate proton profile: gen_values = proton_generation_profile(x_nodes, E_MeV, dose_rate)
   b. Zero generation in p+ substrate (same pattern as cce_vs_bias): set gen_values to 0 where x < junction_pos
   c. Use solve_with_continuation(device_info, gen_values, n_steps=n_continuation_steps)
   d. Extract CCE via compute_cce_from_dd(device_info, gen_values)
   e. Store result
6. Also compute a "no-Auger" reference CCE at the lowest dose rate for comparison (create second device without Auger, solve at same conditions)
7. Clean up: devsim.delete_device(device)
8. Return dict: {"dose_rates": array, "cce_values": array, "cce_no_auger_ref": float, "V_bias": V_bias, "epi_thickness_cm": epi_thickness_cm, "E_MeV": E_MeV}
