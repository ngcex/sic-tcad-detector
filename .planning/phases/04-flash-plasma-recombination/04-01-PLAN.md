  <action>
Create `src/flash_recombination.py` with two main functions:

**1. `add_auger_recombination(device_info)`**

- Set C_n and C_p as devsim region parameters from SiC4H_Parameters (already in device_info["params"])
- Create UAuger node model: `"(C_n * Electrons + C_p * Holes) * (Electrons * Holes - n_i^2)"`
- Create derivatives of UAuger w.r.t. Electrons and Holes using CreateNodeModelDerivative (CRITICAL for Newton Jacobian convergence)
- Update ElectronGeneration to include Auger: `"-ElectronCharge * (USRH + UAuger) + ElectronCharge * RadGenRate"`
- Update HoleGeneration: `"+ElectronCharge * (USRH + UAuger) - ElectronCharge * RadGenRate"`
- Create derivatives of updated ElectronGeneration and HoleGeneration w.r.t. Electrons and Holes
- Re-register continuity equations (same pattern as add_generation_to_dd)
- Set device_info["auger_enabled"] = True
- NOTE: This function MUST be called AFTER add_generation_to_dd (so RadGenRate model exists), OR it must create a zero RadGenRate model if it doesn't exist yet. Check for RadGenRate existence: try to get_node_model_values; if it fails, create "0" model first.

**2. `solve_with_continuation(device_info, generation_values, n_steps=5)`**

- Ramp generation rate from 10% to 100% of target in `n_steps` equal fractional steps
- At each step, scale generation_values by the fraction and call add_generation_to_dd (which updates RadGenRate values in-place)
- Solve DC with relaxed tolerances: absolute_error=1e10, relative_error=1e-10, maximum_iterations=60
- If solve fails at any step, try halving the step size (bisect the remaining ramp)
- Log the fractional step and convergence status at each step
- Return True if converged at full generation, False otherwise

Import pattern: use `from devsim_tools import CreateNodeModel, CreateNodeModelDerivative` (same import as drift_diffusion.py uses). Check drift_diffusion.py imports to match exactly.

Add module docstring explaining the physics: at conventional dose rates, SRH dominates recombination. At FLASH dose rates (20-230 Gy/s), carrier densities may become high enough that Auger recombination (proportional to n^3) becomes significant, reducing CCE.
