**A. Add iv_sweep() to src/drift_diffusion.py:**

```python
def iv_sweep(device_info, V_range, contact="anode", V_step_forward=0.1, V_step_reverse=0.5):
```

- Sweeps voltage over V_range array, extracting contact current at each point
- Uses devsim.set_parameter to set contact bias, then devsim.solve(type="dc")
- For convergence: ramp from current bias to target in small steps (V_step_forward for V>0, V_step_reverse for V<0) per research anti-pattern guidance
- Try primary solve with absolute_error=1e10, relative_error=1e-10, max_iterations=40; on failure, retry with relaxed tolerances (1e12, 1e-8, 100)
- Returns dict with "voltages" and "currents" (numpy arrays)
- Note: currents are in A/cm^2 (1D simulation). Multiply by device area for absolute values.

**B. Create src/cv_analysis.py:**

Constants: EPS_0 = 8.854e-14 F/cm, Q = 1.602e-19 C

Functions:

1. `junction_capacitance(W, eps_r=9.7, area=1.0)` -- returns C = eps_r _ EPS_0 _ area / W (F or F/cm^2)
2. `depletion_width_from_capacitance(C, eps_r=9.7, area=1.0)` -- returns W = eps_r _ EPS_0 _ area / C
3. `compute_cv_from_depletion(voltages, depletion_widths, eps_r=9.7, area=1.0)` -- returns dict with "voltages", "capacitance", "one_over_C_squared" arrays
4. `cv_sweep(device_info, V_range)` -- sweeps reverse bias, extracts W at each voltage using ramp_voltage + extract_depletion_width_numerical from poisson.py, then computes C via junction_capacitance. Returns dict with "voltages", "depletion_widths", "capacitance".
