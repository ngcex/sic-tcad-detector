Create `src/device2d.py` module with a `create_sic_2d_device()` function that:

1. **Function signature** mirrors 1D `create_sic_device()` but adds `half_width_um` parameter:

   ```python
   def create_sic_2d_device(
       device_name="sic2d",
       region_name="sic",
       half_width_um=50.0,       # half-width for symmetry exploitation
       epi_thickness_cm=10e-4,
       substrate_thickness_cm=1e-4,
       N_A=1e19,
       N_D=None,
       T=300,
       doping_profile="graded",  # default to graded (v3.0 decision)
       N_D_junction=None,
       N_D_bulk=None,
       L_transition=None,
   ) -> dict:
   ```

2. **Mesh creation** using `devsim.create_2d_mesh`:
   - Convention: x = lateral (0 to half_width_cm), y = depth (0 to total_depth)
   - Lateral mesh lines: fine near center (ps=5e-5 cm) for symmetry plane, coarser toward edge (ps=5e-4 cm). Add 3-4 intermediate lateral lines.
   - Depth mesh lines: copy the proven 1D spacing from device.py but in y-direction. Fine at junction (ps=1e-7 cm), moderate in epi (ps=5e-6 cm), fine at contacts.
   - **Air buffer regions**: Create thin (1e-8 cm) air buffer at y=-buf and y=total_depth+buf for contact node detection. Use material="SiC" for the buffer regions (same material as main region per research finding).
   - Main SiC region: yl=0, yh=total_depth
   - Contacts: anode at y=0, cathode at y=total_depth, both with bloat=1e-10

3. **Material parameters**: Set the same SiC parameters as 1D (eps_r=9.7, n_i, E_g, mu_n, mu_p, SRH params, kT) using the same `SiC4H_Parameters` and helper functions.

4. **Doping profile** (MESH-04):
   - For "graded" profile: Use the SAME expression as 1D but replace `x` with `y`:
     ```
     Acceptors: N_A_ionized * step(junction_pos - y)
     Donors: (N_D_bulk + (N_D_junction - N_D_bulk) * exp(-max(y - junction_pos, 0) / L_transition)) * step(y - junction_pos)
     NetDoping: Donors - Acceptors
     ```
   - For "uniform" profile: Same x-to-y replacement.
   - Use same calibrated defaults: N_D_junction=2.9e15, N_D_bulk=8.5e13, L_transition=1e-4.

5. **Return** a device_info dict compatible with the 1D pattern (same keys + extras):

   ```python

       "device_name": ..., "region_name": ..., "junction_pos": ...,
       "epi_thickness_cm": ..., "substrate_thickness_cm": ..., "total_length": ...,
       "N_D": ..., "N_A": ..., "N_A_ionized": ..., "T": ..., "n_i": ..., "E_g": ...,
       "params": ..., "mu_n": ..., "mu_p": ..., "num_nodes": ...,
       "doping_profile": ..., "N_D_junction": ..., "N_D_bulk": ..., "L_transition": ...,
       "half_width_cm": ...,   # NEW: lateral half-width in cm
       "dimension": 2,          # NEW: flag for 2D
   }
   ```

6. **Helper function** `set_graded_doping_2d()` and `set_doping_profile_2d()` (private or public) that apply doping using y-coordinate expressions. Keep as module-level functions (same pattern as device.py).
