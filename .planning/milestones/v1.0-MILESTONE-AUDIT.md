## Cross-Phase Integration

**34 exports verified, 0 missing connections, 0 broken flows.**

Key integration chains:

1. **Material → Device → DD → CCE → FLASH → Parametric**: Complete chain from `SiC4H_Parameters` through all simulation stages
2. **Graded doping calibration**: Phase 2 values (N_D_junction=2.90e15, N_D_bulk=8.50e13) used consistently in Phases 3-5
3. **Generation profiles → FLASH**: `proton_generation_profile` consumed by `cce_vs_dose_rate`
4. **Auger model → Parametric**: `add_auger_recombination` wired into parametric sweep via `cce_vs_dose_rate`
5. **Centralized constants (Phase 6)**: `SiC4H_Parameters` replaces hardcoded values in `generation_profiles.py` and `charge_collection.py`
6. **time_node_model fix (Phase 7)**: Present in all 3 equation registration sites (`drift_diffusion.py`, `charge_collection.py`, `flash_recombination.py`)

### Notable Observations (non-blocking)

1. **Calibrated doping duplication**: N_D_junction/N_D_bulk/L_transition appear as literals in 4 locations (charge_collection.py x2, flash_recombination.py, notebook 02). Values are consistent but not centralized.
2. **Deferred imports**: `charge_collection.py` uses function-body imports for `drift_diffusion.py` symbols to avoid circular dependencies. Sound pattern but import errors surface at runtime.
3. **compute_ni() orphaned**: Defined in `sic_material.py`, tested, but unused in v1. Documented as v2-only (ADV-02).
