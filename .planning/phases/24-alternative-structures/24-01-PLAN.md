  <action>
Create src/alternative_structures.py with four public mesh builder functions. Each returns a device_info dict compatible with the existing pipeline (matching the contract from device2d.py plus a `structure_type` key).

**Shared infrastructure (internal helpers):**

- `_set_sic_material_params(device_name, region_name, N_D, N_A, T)` — replicates the material parameter setup from device2d.py (Permittivity, ElectronCharge, n_i, T, kT, V_t, mu_n, mu_p, SRH params). Returns a partial device_info dict with material keys.
- `_set_doping_and_build_info(device_name, region_name, junction_pos, epi_thickness_cm, substrate_thickness_cm, N_A, N_D, T, doping_profile, N_D_junction, N_D_bulk, L_transition, half_width_cm, material_info, extra_keys)` — sets doping, logs, returns complete device_info dict.
- Import doping helpers from device2d.py: `set_doping_profile_2d`, `set_graded_doping_2d`.
- Use the same physical constants as device2d.py (EPS_0, Q, K_B_EV, K_B_J).
- Use the same default doping values as device2d.py (\_N_D_JUNCTION_DEFAULT, etc.).

**Function 1: create_mesa_device()**

- Signature: `create_mesa_device(device_name="mesa2d", region_name="sic", pillar_half_width_um=50.0, trench_width_um=10.0, trench_depth_um=10.0, epi_thickness_cm=10e-4, substrate_thickness_cm=1e-4, N_A=1e19, T=300, doping_profile="graded", N_D_junction=None, N_D_bulk=None, L_transition=None)`
- Mesa structure: SiC pillar (active SV) with trenches cut on both sides. Uses mirror symmetry at x=0 (half-device).
- Lateral layout: [0, pillar_hw] = SiC pillar, [pillar_hw, pillar_hw + trench_w] = trench fill.
- Trench fill: Use SiC material with zero doping (same air buffer trick as device2d.py — avoids interface equation complications). Set zero doping by creating a separate node model with Donors=0 and Acceptors=0 in the trench region.
- Trench extends from y=0 to min(trench_depth, total_depth). If trench is shallower than total depth, substrate below trench is a separate region with bulk substrate doping.
- Contacts: anode at y=0 on pillar region only, cathode at y=total_depth across full width.
- Air buffer regions at top and bottom (same 1e-8 cm pattern as device2d.py).
- Returns device_info with `structure_type="mesa"`.

**Function 2: create_3d_electrode_device()**

- Signature: `create_3d_electrode_device(device_name="elec3d", region_name="sic", outer_radius_um=50.0, column_radius_um=5.0, epi_thickness_cm=10e-4, substrate_thickness_cm=1e-4, N_A=1e19, T=300, doping_profile="graded", N_D_junction=None, N_D_bulk=None, L_transition=None)`
- Axisymmetric 3D electrode: x=radial (r), y=depth (z). Central n+ column at r=0 to column_radius as cathode. Outer p+ at r=outer_radius as anode.
- Mesh: fine radial spacing near column (1e-6 cm), intermediate at column edge, coarser toward outer radius.
- CRITICAL: After create_device and BEFORE any physics setup, activate cylindrical coordinates:
  ```python
  devsim.set_parameter(name="raxis_zero", value=0.0)
  devsim.set_parameter(name="raxis_variable", value="x")
  for region in all_regions:
      devsim.cylindrical_node_volume(device=device_name, region=region)
      devsim.cylindrical_edge_couple(device=device_name, region=region)
      devsim.cylindrical_surface_area(device=device_name, region=region)
  devsim.set_parameter(name="node_volume_model", value="CylindricalNodeVolume")
  devsim.set_parameter(name="edge_couple_model", value="CylindricalEdgeCouple")
  devsim.set_parameter(name="element_edge_couple_model", value="ElementCylindricalEdgeCouple")
  devsim.set_parameter(name="element_node0_volume_model", value="ElementCylindricalNodeVolume@en0")
  devsim.set_parameter(name="element_node1_volume_model", value="ElementCylindricalNodeVolume@en1")
  ```
- Doping: p+ substrate at y < junction_pos, n- epi at y >= junction_pos (same as planar, but now represents radial cross-section).
- For the central n+ column: add high donor doping (1e19) at x < column_radius via node model expression.
- Contacts: cathode at x=0..column_radius, y=0 (top of central column). Anode at x=outer_radius, full y range (outer surface).
- Returns device_info with `structure_type="3d_electrode"`, `coordinate_system="cylindrical"`, `outer_radius_cm`, `column_radius_cm`.

**Function 3: create_delta_e_e_device()**

- Signature: `create_delta_e_e_device(device_name="dee2d", region_name_de="delta_e", region_name_e="e_stop", half_width_um=50.0, delta_e_thickness_um=2.0, e_stop_thickness_um=50.0, substrate_thickness_cm=1e-4, N_A=1e19, T=300, doping_profile="graded", N_D_junction=None, N_D_bulk=None, L_transition=None)`
- Two stacked SiC regions with devsim interface between them.
- delta-E layer: thin (default 2 um), for particle identification.
- E-stop layer: thick (default 50 um), for total energy measurement.
- Use `devsim.add_2d_interface()` to connect delta_e and e_stop regions with continuous potential.
- Each layer has its own anode-cathode contact pair: `de_anode`, `de_cathode`, `estop_anode`, `estop_cathode`.
- Both layers get the same SiC material params and graded doping.
- Returns device_info with `structure_type="delta_e_e"`, plus `region_name_de`, `region_name_e`, `delta_e_thickness_cm`, `e_stop_thickness_cm`.

**Function 4: create_guard_ring_device()**

- Signature: `create_guard_ring_device(device_name="gr2d", region_name="sic", sv_half_width_um=50.0, guard_ring_width_um=5.0, guard_ring_gap_um=3.0, guard_ring_depth_um=1.0, epi_thickness_cm=10e-4, substrate_thickness_cm=1e-4, N_A=1e19, N_A_guard=5e18, T=300, doping_profile="graded", N_D_junction=None, N_D_bulk=None, L_transition=None)`
- Extended planar device with a p+ guard ring beyond the SV edge.
- Lateral layout (half-device): [0, sv_hw] = SV active area, [sv_hw, sv_hw+gap] = transition, [sv_hw+gap, sv_hw+gap+gr_w] = guard ring.
- Guard ring: additional p+ implant defined by node model expression adding acceptor doping (N_A_guard) at the guard ring lateral position and depth.
- Guard ring contact: separate metal contact at guard ring surface for independent current extraction.
- Main contacts: anode at y=0 on SV (x < sv_hw), cathode at y=total_depth.
- Returns device_info with `structure_type="guard_ring"`, `guard_ring_contact="guard_ring_anode"`, `sv_half_width_cm`, `total_half_width_cm`.

**Also add a convenience function:**

- `restore_cartesian_coords()` — resets global devsim parameters to Cartesian defaults (NodeVolume, EdgeCouple, etc.) after deleting a cylindrical device. Called by users who need to create a Cartesian device after a cylindrical one.

**Important implementation notes:**

- Do NOT modify any existing module (device2d.py, poisson.py, drift_diffusion.py, etc.).
- All structures use the same SiC material setup as device2d.py (import and reuse the constants and material functions).
- Each builder function should be self-contained: create mesh, set material params, set doping, return device_info.
- Follow the established logging pattern from device2d.py.
