Implementation sequence per fluence point (staged creation + TAT):

- `compute_damaged_params(pristine_tau_n, pristine_tau_p, pristine_N_D_profile, fluence, ...)`
- `create_sic_device(device_name=f"dc_fluence_{uuid}", epi_thickness_cm=..., doping_profile="graded", N_D_junction=2.90e15, N_D_bulk=8.50e13, L_transition=1.0e-4)`
- `apply_damaged_params(device_info, damaged)`
- `setup_poisson(device_info)` + `solve_equilibrium(device_info)`
- `setup_sic_drift_diffusion(device_info)` + set `device_info["dd_initialized"] = True`
- `setup_tat_model(device_info, N_t=N_t)` + `setup_surface_recombination(device_info, S_n=S_n, S_p=S_p)`
- `ramp_bias(device_info, V_bias, contact="anode", V_step=1.0)` — note: bias on ANODE (negative = reverse), matching existing `dark_current_sweep()` convention
- `_compute_node_efield(device, region)` + `_compute_gamma_factors(device, region)` — CRITICAL: must recompute after bias ramp
- `extract_dark_current_components(device_info, area=area)`

Extract pristine parameters ONCE before the loop (same as cce_vs_fluence): create a reference device, get pristine tau_n/tau_p from `srh_lifetime(300.0, "electron"/"hole")`, get pristine N_D_profile from epi-region Donors, then delete the reference device.

Return dict with keys: `fluences`, `I_total`, `I_SRH`, `I_TAT`, `I_SRV` (all np arrays), `I_baseline` (float, I_total at first fluence point if it is 0.0), `delta_I` (I_total - I_baseline, only if first fluence is 0.0), `V_bias`, `energy_MeV`, `lifetime_model`.

Error handling: wrap each fluence point in try/except, set all components to NaN on failure, always delete device in finally block. Log progress with `logger.info(f"dark_current_vs_fluence: fluence={fluence:.2e}, I_total={...:.3e} A")`.

Use `import devsim.python_packages.simple_physics as simple_physics` and `simple_physics.GetContactBiasName("anode")` for bias ramping — match the existing `dark_current_sweep()` pattern at line 407 of dark_current.py.
