  <files>src/generation_profiles.py, tests/test_generation_profiles.py</files>
  <action>
Create src/generation_profiles.py with these functions:

1. `dose_rate_to_generation(dose_rate_Gy_s, rho_g_cm3=3.21, E_pair_eV=8.4)`:
   Convert dose rate (Gy/s) to generation rate (cm^-3 s^-1).
   Formula: G = dose_rate _ rho _ 1e4 / (E_pair_eV \* 1.602e-12)
   where 1 Gy = 1e4 erg/g in CGS.

2. `alpha_generation_profile(x_cm, alpha_range_cm=15e-4, total_energy_eV=5.486e6, E_pair_eV=8.4)`:
   Generate spatial e-h pair generation profile for Am-241 alpha particles.
   - Use a smooth profile: approximately uniform from 0 to ~0.8*range, then smooth erfc roll-off (width ~0.1*range) to avoid numerical ringing.
   - Normalize so that integral(G(x)\*dx) = total_energy_eV / E_pair_eV (total pairs generated per alpha).
   - Return G(x) in units of cm^-1 (pairs per cm depth per incident alpha). The caller multiplies by fluence rate to get cm^-3 s^-1.
   - Alpha range in SiC: use default 15 um (15e-4 cm) per research, with parameter to override.

3. `proton_generation_profile(x_cm, E_MeV, dose_rate_Gy_s=1.0)`:
   Generate carrier generation rate for proton beams at 30, 70, 150 MeV.
   - Hardcode NIST PSTAR CSDA ranges in water: {30: 8.85, 62: 31.0, 70: 40.8, 150: 157.7} mm.
   - Scale to SiC via density: R_SiC = R_water \* (1.0 / 3.21).
   - For all therapeutic energies, range >> 10 um detector, so profile within detector is flat (entrance dose region).
   - Return G(x) in cm^-3 s^-1 using dose_rate_to_generation for the magnitude.
   - Support arbitrary E_MeV by interpolation of range table if between tabulated values.

4. Module-level constants: RHO_SIC = 3.21, E_PAIR_SIC_EV = 8.4, PROTON_RANGE_WATER_MM dict.

Create tests/test_generation_profiles.py:

- test_dose_rate_conversion: verify dimensional analysis (1 Gy/s in SiC produces ~2.39e21 pairs/cm^3/s)
- test_alpha_profile_normalization: integral of profile equals total_energy/E_pair
- test_alpha_profile_smooth_rolloff: no discontinuity at range boundary (max gradient bounded)
- test_proton_profile_flat_in_detector: for 70 MeV protons, profile variation within 10 um < 1%
- test_proton_range_scaling: R_SiC < R_water for all energies (density > 1)
- test_proton_range_values: spot-check 70 MeV range in SiC ~ 12.7 mm

<task type="auto">
  <name>Task 2: Hecht equation and CCE utilities</name>
  <files>src/charge_collection.py, tests/test_charge_collection.py</files>
  <action>
Create src/charge_collection.py with these functions:

1. `hecht_cce(V, d, mu_e=950.0, tau_e=1e-9, mu_p=125.0, tau_p=6e-7)`:
   Two-carrier Hecht equation implementation per research code example.
   - V: applied voltage (use abs value for reverse bias). Float or numpy array.
   - d: active region thickness (cm).
   - lambda_e = mu_e _ tau_e _ |V| / d, similarly lambda_h.
   - CCE = (lambda_e/d)_(1-exp(-d/lambda_e)) + (lambda_h/d)_(1-exp(-d/lambda_h))
   - Handle V=0 gracefully (return 0.0). Use np.where with np.errstate for vectorized safety.
   - Clip to [0, 1].
   - Default mu/tau values are SiC low-doping literature values from SiC4H_Parameters.

2. `compute_cce_from_current(I_collected, I_generated)`:
   Simple ratio CCE = |I_collected| / I_generated, clipped to [0, 1].
   Used by Plan 02 to extract CCE from DD solution.

3. `hecht_cce_partial_depletion(V, d_epi, W_func, mu_e=950.0, tau_e=1e-9, mu_p=125.0, tau_p=6e-7, L_diff_p=7e-4)`:
   Extended Hecht for partially-depleted detector.
   - W_func: callable W(V) returning depletion width at voltage V (from C-V data or DD).
   - Drift component: standard Hecht with d=W(V) for charge in depletion region.
   - Diffusion component: exp(-x/L_diff) collection probability for charge in neutral region (W to d_epi).
   - L_diff_p default 7 um = 7e-4 cm from literature (CCE simulation references in research).
   - Document that this is an approximation for comparison purposes.

Create tests/test_charge_collection.py:

- test_hecht_zero_voltage: CCE = 0 at V=0

- test_hecht_monotonic: CCE increases monotonically with voltage
- test_hecht_vectorized: works with numpy array input
- test_hecht_clipped: never exceeds 1.0
- test_cce_from_current_ratio: simple ratio test
- test_hecht_partial_depletion_higher_than_drift_only: diffusion adds to collection
- test_hecht_symmetry: CCE same for +V and -V (uses abs)
