**1. In src/radiation_damage.py**, add `compute_phi_crit()` after the existing `compute_damaged_params()` function:

```python
def compute_phi_crit(N_D_profile, eta_removal=5.0, energy_MeV=62.0, niel_table=None):
```

- Computes Phi_crit (protons/cm^2) = min(N_D_profile where N_D > 0) / (eta_removal \* kappa)
- kappa from `get_hardness_factor(energy_MeV, niel_table)`
- Returns a dict with keys: `phi_crit_proton` (protons/cm^2), `phi_crit_neq` (neq/cm^2), `N_D_min` (cm^-3), `kappa`
- For the Petringa device at 62 MeV: Phi_crit ~ 1.39e14 protons/cm^2 (N_D_bulk=8.5e13, eta=5.0, kappa=0.35)
- Uses `RadiationDamageParams().eta_removal` as default eta if not provided; alternatively accept eta_removal directly for simplicity.

**2. In src/cv_analysis.py**, add `cv_at_fluence()` and `plot_cv_evolution()` after the existing `cv_sweep()` function.

`cv_at_fluence()` signature:

```python
def cv_at_fluence(fluence, V_range, area=1.0, epi_thickness_cm=10e-4,
                  energy_MeV=62.0, lifetime_model="linear", damage_params=None,
                  phi_crit_threshold=0.90):
```

Implementation follows the staged-creation pattern from Phases 14/15:

1. Extract pristine params: `srh_lifetime(300, "electron")`, `srh_lifetime(300, "hole")`
2. Create device via `create_sic_device(device_name=uuid, doping_profile="graded", ...)`
3. Extract pristine N_D_profile from epi nodes (same approach as `cce_vs_fluence` and `dark_current_vs_fluence`)
4. Call `compute_phi_crit()` on pristine profile; warn if fluence >= phi_crit_threshold \* phi_crit
5. If fluence >= phi_crit, return None (cannot solve)
6. If fluence > 0: compute `damaged = compute_damaged_params(...)`, call `apply_damaged_params(device_info, damaged)`
7. `setup_poisson()` -> `solve_equilibrium()` -> `setup_sic_drift_diffusion()`
8. `cv_result = cv_sweep(device_info, V_range, area=area)`
9. Cleanup device in `finally` block via `devsim.delete_device(device=dev_name)`
10. Return cv_result dict augmented with `fluence` key

Add imports at top of cv_analysis.py: `uuid`, `from src.device import create_sic_device, apply_damaged_params`, `from src.radiation_damage import compute_damaged_params, compute_phi_crit`, `from src.poisson import setup_poisson, solve_equilibrium`, `from src.drift_diffusion import setup_sic_drift_diffusion`, `from src.sic_material import srh_lifetime`.
