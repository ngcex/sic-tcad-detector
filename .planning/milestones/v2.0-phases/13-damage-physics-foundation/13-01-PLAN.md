Create src/radiation_damage.py as a pure-Python module (NO devsim imports). Follow the project's CGS unit convention and dataclass pattern from sic_material.py.

The module must contain:

1. **RadiationDamageParams dataclass** with provenance-tagged constants from Burin et al. (2024):
   - Z1/2: eta=5.0 cm^-1, E=0.67 eV, sigma_n=2e-14 cm^2, sigma_p=3.5e-14 cm^2, acceptor
   - EH6/7: eta=1.6 cm^-1, E=1.60 eV, sigma_n=9e-12 cm^2, sigma_p=3.8e-14 cm^2, donor
   - EH4: eta=2.4 cm^-1, E=1.03 eV, sigma_n=5e-13 cm^2, sigma_p=5e-14 cm^2, acceptor
   - eta_removal=5.0 cm^-1 (effective carrier removal rate)
   - source="Burin et al., arXiv:2407.16710 (2024)", reference_particle="1 MeV neutron equivalent"
   - Add **post_init** validation: all eta values > 0, all sigma values > 0

2. **NIEL_HARDNESS_PROTON_SIC dict** mapping proton energy (MeV) to hardness factor:
   - {30: 0.50, 62: 0.35, 70: 0.33, 150: 0.22} (placeholder values per RESEARCH.md)
   - Add a module-level docstring noting these are placeholders pending SR-NIEL lookup

3. **Pure functions** (all stateless, taking explicit parameters):
   - `defect_concentration(eta, fluence) -> float`: Returns eta \* fluence
   - `defect_concentrations(params, fluence_neq) -> dict`: Returns dict with N_Z12, N_EH67, N_EH4
   - `degraded_lifetime(tau_0, K_tau, fluence, model="linear") -> float`:
     Linear: 1/tau = 1/tau_0 + K_tau _ fluence, return max(1/inv_tau, 1e-15)
     Logarithmic: tau = tau_0 / (1 + K_tau _ tau_0 \* fluence)^alpha with alpha=0.8 configurable
     Raise ValueError for unknown model
   - `effective_doping(N_D, eta, fluence) -> float`: max(N_D - eta \* fluence, 0.0)
   - `apply_carrier_removal(N_D_profile, eta_removal, fluence_neq, floor=0.0) -> np.ndarray`:
     Position-dependent: np.maximum(N_D_profile - eta_removal \* fluence_neq, floor)
   - `compute_K_tau(params, carrier="electron", T=300.0) -> float`:
     K_tau = sum(eta_i _ sigma_i _ v_th) for all 3 defects
     v_th = sqrt(3 _ k_B _ T / m_eff) \* 100 (m/s -> cm/s)
     Use m_e_dos=0.77, m_h_dos=1.0 from SiC4H_Parameters (import the constants, don't duplicate)
   - `scale_to_proton_energy(damage_constant, energy_MeV, niel_table=None) -> float`:
     Uses np.interp for interpolation between table entries
   - `get_hardness_factor(energy_MeV, niel_table=None) -> float`:
     Wrapper returning kappa for given energy

4. **High-level interface** `compute_damaged_params(...)`:

   ```
   compute_damaged_params(
       pristine_tau_n, pristine_tau_p, N_D_profile,
       fluence, energy_MeV=62.0, damage_params=None,
       lifetime_model="linear", T=300.0
   ) -> dict
   ```

   CRITICAL: Short-circuit at fluence <= 0 -- return pristine values with NO arithmetic operations (regression safety per Pitfall 1 in RESEARCH.md).
   Otherwise: convert to neq equivalent, compute K_tau for electrons and holes, compute degraded lifetimes, apply carrier removal, compute defect concentrations.
   Return dict with keys: tau_n, tau_p, N_D_profile, N_Z12, N_EH67, N_EH4, fluence, fluence_neq, energy_MeV, lifetime_model.

5. **Logging**: Use `logging.getLogger(__name__)` pattern. Log warning when effective doping at any position drops below 1e10 cm^-3 (approaching compensation).

Import only: numpy, logging, dataclasses. Do NOT import devsim anywhere in this file.
