**2. estimate_noise_floor()** (FEAS-03):

```python
def estimate_noise_floor(
    I_dark_A,
    t_shaping_s=1e-6,
    sv_thickness_um=10.0,
    sv_width_um=100.0,
    sv_depth_um=None,
    k_sigma=3.0,
) -> dict:
```

- Pure computation function (no device creation). Takes dark current as input.
- Q = 1.602e-19 (electron charge)
- E_pair_eV = 8.4 (SiC pair creation energy)
- sigma_shot = sqrt(2 _ Q _ abs(I_dark_A) \* t_shaping_s)
- Q_min = k_sigma \* sigma_shot
- E_min_eV = Q_min \* E_pair_eV / Q
- E_min_keV = E_min_eV / 1e3
- l_bar from mean_chord_length(sv_thickness_um, sv_width_um, sv_depth_um)
- y_min_keV_um = E_min_keV / l_bar
- Returns dict: I_dark_A, sigma_shot_C, Q_min_fC (Q_min\*1e15), E_min_keV, y_min_keV_um, l_bar_um, t_shaping_s, k_sigma
- Add docstring noting this is detector-intrinsic shot noise only; real systems will have higher noise from readout electronics

**3. score_structures()** (FEAS-02):

```python
def score_structures(
    metrics_dict: dict,
    weights: dict = None,
) -> pd.DataFrame:
```

- metrics_dict: {structure_name: {cce_uniformity, noise_floor, spectral_resolution, fabrication_complexity}}
  - cce_uniformity: edge/center ratio (higher = better)
  - noise_floor: y_min in keV/um (lower = better)
  - spectral_resolution: y_D/y_F ratio or std of d(y) peak (lower = better)
  - fabrication_complexity: 1-4 scale (1=planar, 2=guard_ring, 3=mesa, 4=3d_electrode/delta_E_E) (lower = better)
- Default weights: cce_uniformity=0.30, noise_floor=0.20, spectral_resolution=0.20, fabrication_complexity=0.30
- Normalize each metric to [0, 1] across structures. For "higher is better" metrics (cce_uniformity), normalized = (val - min) / (max - min). For "lower is better" metrics (noise, spectral_res, fab_complexity), normalized = 1 - (val - min) / (max - min). Handle single-value case (all equal -> 0.5).
- weighted_score = sum(weight_i \* normalized_i)
- Returns DataFrame with columns: structure, each raw metric, each normalized metric, weighted_score. Sorted by weighted_score descending.
